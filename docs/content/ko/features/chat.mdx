# AI 채팅

AI 채팅은 ClaudeShip으로 애플리케이션을 구축하기 위한 주요 인터페이스입니다. 자연어로 원하는 것을 설명하면, AI가 코드를 작성합니다.

## 작동 방식

```
┌─────────────────────────────────────┐
│  사용자: "로그인 페이지 만들어줘"     │
├─────────────────────────────────────┤
│  AI: 로그인 페이지 생성 중...        │
│      • 사용자명/이메일 입력          │
│      • 비밀번호 필드                 │
│      • 제출 버튼                     │
│      • 폼 유효성 검사                │
└─────────────────────────────────────┘
```

1. 채팅 입력창에 요청을 입력하세요
2. AI가 분석하고 코드를 생성합니다
3. 변경 사항이 라이브 프리뷰에 반영됩니다
4. 만족할 때까지 계속 반복하세요

## 기능

### 스트리밍 응답

스트리밍 메시지 블록으로 AI가 실시간으로 코드를 작성하는 것을 지켜보세요. 각 단계가 일어나는 것을 바로 확인할 수 있습니다.

### 대화 기억

채팅 기록이 자동으로 저장됩니다. 컨텍스트를 다시 설명할 필요 없이 중단한 곳부터 계속하세요.

### 도구 사용

AI는 작업을 완료하기 위해 다양한 도구를 사용할 수 있습니다:
- **Write**: 파일 생성 또는 수정
- **Bash**: 터미널 명령 실행
- **Read**: 파일 내용 확인
- **Search**: 패턴으로 파일 찾기

### 사용자 질문

필요할 때 AI가 질문합니다:
- 여러 옵션 중 선택
- 파괴적인 작업 확인
- 사용자 정의 입력 제공

## 예시 프롬프트

### 프론트엔드

```text
"드롭다운 메뉴가 있는 반응형 네비게이션 바 만들어줘"
"부드러운 전환 효과가 있는 다크 모드 토글 추가해줘"
"호버 효과가 있는 카드 그리드 레이아웃 빌드해줘"
```

### 백엔드

```text
"사용자 인증을 위한 REST API 만들어줘"
"블로그 게시물을 위한 데이터베이스 스키마 추가해줘"
"파일 업로드 엔드포인트 구현해줘"
```

### 풀스택

```text
"마크다운 지원이 있는 완전한 블로그 만들어줘"
"차트와 필터가 있는 대시보드 생성해줘"
"실시간 채팅 애플리케이션 만들어줘"
```

## 새 대화

채팅 헤더의 **+ New** 버튼을 클릭하여 새로 시작하세요. 이전 대화는 프로젝트에 보존됩니다.

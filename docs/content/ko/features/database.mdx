# 데이터베이스 뷰어

UI에서 직접 SQLite 데이터베이스를 찾아보고 관리하세요. 테이블을 보고, SQL 쿼리를 실행하며, ClaudeShip을 떠나지 않고 데이터를 편집하세요.

## 기능

### 테이블 브라우저

- 데이터베이스의 모든 테이블 보기
- 각 테이블의 행 수 표시
- 페이지네이션으로 테이블 데이터 찾아보기

### 데이터 그리드

각 테이블에 대해 다음을 확인하세요:
- 열 이름 및 유형
- 기본 키 표시기
- NULL 값 처리
- 페이지네이션된 결과 (페이지당 50행)

### SQL 쿼리 에디터

사용자 정의 SQL 쿼리 실행:
1. 헤더에서 **SQL** 버튼 클릭
2. 쿼리 입력
3. **Cmd+Enter** 누르거나 **Run** 클릭

```sql
SELECT * FROM users WHERE email LIKE '%@gmail.com'
SELECT COUNT(*) FROM posts WHERE status = 'published'
```

### 행 삭제

휴지통 아이콘으로 개별 행을 삭제하세요. 식별을 위해 기본 키가 사용됩니다.

## 데이터베이스 지원

ClaudeShip은 현재 **SQLite** 데이터베이스를 지원하며, 다음 구성 사용 시 자동으로 생성됩니다:
- Express + Prisma 구성
- FastAPI + SQLAlchemy 구성

## 키보드 단축키

| 단축키 | 동작 |
|----------|--------|
| Cmd+Enter | SQL 쿼리 실행 |

## 사용법

1. 오른쪽 패널에서 **Database** 탭 클릭
2. 사이드바에서 테이블 선택
3. 데이터 찾아보기 또는 SQL 모드로 전환
4. 필요에 따라 쿼리 실행

## 팁

- 복잡한 쿼리에는 SQL 모드 사용
- 삭제 작업에는 확인 필요
- 변경 사항이 즉시 반영됨
- 새로고침 버튼으로 테이블 목록 다시 로드

# 파일 탐색기

트리 뷰에서 프로젝트 구조를 찾아보세요. 파일을 클릭하면 구문 강조가 적용된 내용을 확인할 수 있습니다.

## 기능

### 트리 뷰

- 계층형 폴더 구조
- 폴더 확장/축소
- 파일 유형 아이콘 (TS, JS, Python 등)
- 확장자별 색상 코드

### 파일 뷰어

파일을 클릭하면 모달이 열리고 다음을 표시합니다:
- 전체 파일 내용
- 구문 강조
- 파일 경로 표시
- 닫기 버튼

### 자동 새로고침

파일 탐색기는 AI가 파일을 생성하거나 수정할 때 자동으로 새로고침됩니다.

## 파일 아이콘

| 확장자 | 색상 | 아이콘 |
|-----------|-------|------|
| `.ts`, `.tsx` | 파랑 | 파일 |
| `.js`, `.jsx` | 노랑 | 파일 |
| `.py` | 녹색 | 파일 |
| `.json` | 노랑 | 파일 |
| `.css` | 보라 | 파일 |
| `.html` | 주황 | 파일 |
| `.prisma` | 청록 | 파일 |
| `.md` | 회색 | 파일 |

## 키보드 단축키

| 단축키 | 동작 |
|----------|--------|
| 파일 클릭 | 내용 보기 |
| 폴더 클릭 | 확장/축소 |
| 새로고침 버튼 클릭 | 트리 다시 로드 |

## 사용법

1. 왼쪽 사이드바의 폴더 아이콘을 클릭하여 파일 탐색기 토글
2. 폴더를 클릭하여 탐색
3. 파일을 클릭하여 내용 확인
4. 새로고침 버튼을 클릭하여 트리 다시 로드

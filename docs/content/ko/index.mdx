# ClaudeShip 문서에 오신 것을 환영합니다

ClaudeShip은 **AI 기반 개발 환경**으로, 자연어로 웹 애플리케이션을 구축할 수 있습니다.

코드를 한 줄씩 작성하는 대신, 원하는 것을 설명하기만 하면 됩니다. AI가 코드를 생성하고, 패키지를 설치하며, 앱을 실시간으로 실행합니다.

## 빠른 링크

- [시작하기](/docs/getting-started) - ClaudeShip 설치 및 설정
- [기능](/docs/features) - 모든 기능 살펴보기
- [채팅](/docs/features/chat) - AI 채팅 사용법

## 주요 기능

- **자연어 개발** - 평범한 영어로 원하는 것을 설명하세요
- **라이브 프리뷰** - 변경 사항을 실시간으로 확인
- **풀스택 지원** - 프론트엔드 + 백엔드 + 데이터베이스
- **Git 기반 체크포인트** - 원하는 상태로 쉽게 복구
- **브라우저 테스트** - Playwright로 자동화된 E2E 테스트

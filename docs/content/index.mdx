# Welcome to ClaudeShip Docs

ClaudeShip is an **AI-powered development environment** that lets you build web applications using natural language.

Instead of writing code line by line, simply describe what you want to build. The AI generates code, installs packages, and runs your app in real-time.

## Quick Links

- [Getting Started](/docs/getting-started) - Set up ClaudeShip on your machine
- [Features](/docs/features) - Explore all capabilities
- [Chat](/docs/features/chat) - Learn how to use AI chat

## Key Features

- **Natural Language Development** - Describe what you want in plain English
- **Live Preview** - See changes in real-time
- **Full-Stack Support** - Frontend + Backend + Database
- **Git-Based Checkpoints** - Easy rollback to any state
- **Browser Testing** - Automated E2E testing with Playwright
